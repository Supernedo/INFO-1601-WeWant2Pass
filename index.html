PROJECT INFORMATION

The Group (A Sip of Java), formerly called WeWant2Pass has established a website to generate the Weather for any City in the world.

This website uses the API from Open Weather Map to pull and display data about the current weather from any city.

The website starts by determining whether or not your browser supports geolocation, once true, it will provide current statistics based on your location accurate to the minute.

Yup, the part you've all been waiting for, the masterminds behind the genius of the site. Okay maybe not genius but, the website was developed by Ché Salazar, Seth Colon and Darius Castillo with the aim of proper Data Visualization.
